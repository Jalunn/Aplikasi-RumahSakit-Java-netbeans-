# Aplikasi Rumah Sakit (Java + MySQL)

Aplikasi desktop untuk manajemen layanan rumah sakit menggunakan Java Swing dan MySQL.

## Fitur:
- Login user
- CRUD data dokter, apotek, operasi, dan pembayaran
- Dashboard GUI interaktif

## Teknologi:
- Java Swing
- JDBC
- MySQL
- NetBeans IDE
